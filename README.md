# DriveLine 

Car showroom website made with HTML, CSS, JavaScript, and `fetch()`.

## Features

- fetches car inventory from `cars.json` using the Fetch API
- displays a premium car gallery with  car name, brand, body type, price, mileage, and distance driven
- sorts cars by  price, mileage, brand, and category
- filters cars by category, brand, condition, distance driven, and price range
- searches cars by brand or car name
- uses clean showroom branding instead of placeholder API wording

## Project Files

- `index.html` for the showroom page
- `script.js` for fetching, sorting, and filtering
- `styles.css` for styling
- `cars.json` for inventory data

## Run locally

Run the project with a local server, then open `index.html`.
